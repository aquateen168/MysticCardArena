# 🚀 PokéCollector • Modern Sets (2021+)

**Your personal Pokémon TCG collection tracker with live prices, checklist, and news feed.**

A beautiful, fast, browser-based web app built just for modern Pokémon cards (2021 and newer). Mark what you own, see real TCGPlayer market prices, and track your collection value instantly.

### ✨ Features

- ✅ **Live TCGPlayer Market Prices** shown on every card  
- ✅ **Separate rows for every variant** (Normal, Reverse Holo, Holo Rare, etc.)  
- ✅ **Real-time total collection value** (updates automatically)  
- ✅ **Full checklist** with owned / missing filters  
- ✅ **Click any card** → modal with variant toggle + exact PriceCharting link  
- ✅ **PokéBeach latest news** feed  
- ✅ **The Card Arena** latest YouTube videos  
- ✅ Export / Import your collection (JSON backup)  
- ✅ One-click **Reset Collection** button  
- ✅ Fully responsive (works great on mobile)  
- ✅ 100% client-side — no server needed

### 🛠️ How to Use

1. Open the site  
2. Pick a modern set (2021+) from the dropdown  
3. Click the + on any card to mark it owned  
4. Prices and total value update automatically  
5. Click any card for full details and variant switching
